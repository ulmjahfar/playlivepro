## Image Guidelines

1. **Format:** JPG or PNG
2. **Quality:** High quality, optimized for web (compressed but clear)
3. **Style:** Modern, professional, consistent color scheme matching PlayLive brand
4. **Colors:** Should complement the dark theme (#050b1d background) with purple/blue accents
5. **Fallback:** Images gracefully hide if missing (onError handlers in place)

## Notes

- All images are optional - the homepage will work without them
- Images should be generated using AI tools (DALL-E, Midjourney, Stable Diffusion, etc.)
- Consider generating images with consistent style and color palette
- Images will be displayed with hover effects and transitions
